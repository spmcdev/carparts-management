🔗 API Usage Examples

Get all sold stock:
GET /sold-stock-report
Filter by container:
GET /sold-stock-report?container_no=CNT001
Local purchases only:
GET /sold-stock-report?local_purchase=true
Date range with pagination:
GET /sold-stock-report?from_date=2024-01-01&to_date=2024-12-31&page=1&limit=20
Combined filters:
GET /sold-stock-report?container_no=CNT001&local_purchase=false&from_date=2024-01-01
Summary statistics only:
GET /sold-stock-summary?local_purchase=true

🚀 How to Use

  1. Start the server: node index.js
  2. Open the frontend and navigate to Stock Management
  3. Use the enhanced filters:
    • Select container numbers from dropdown
    • Choose purchase type (Local/Container/All)
    • Set optional date ranges
  4. View comprehensive analytics with summary cards and detailed tables
  5. Print enhanced reports with all the new data fields
💡 Pro Tip: All filters are optional and can be combined for precise reporting. The system now provides real-time profit margin calculations when cost price data is available.
